# 🌦️ Weather App

A simple and responsive **Weather Application** built using **HTML, CSS, and JavaScript**.  
This app fetches real-time weather data using an API and displays current weather conditions.

---

## 📌 Live Demo

🌐 https://weather-app-psi-seven-99.vercel.app

---

## 💻 Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Weather API (OpenWeather or similar)

---

## 🎯 Features

- 🌍 Search any city weather  
- 🌡️ Real-time temperature updates  
- 🌤️ Weather condition display (sunny, cloudy, rain, etc.)  
- 📱 Responsive design (mobile-friendly)  
- ⚡ Fast API response handling  

---

## 📁 Project Structure

- `index.html` → Main UI  
- `style.css` → Styling file  
- `app.js` → API logic + functionality  

---

## 🚀 How It Works

1. User enters city name  
2. App sends request to weather API  
3. API returns weather data  
4. Data is displayed on UI
